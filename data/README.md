# Data Sources

Synthetic source-style data for benefits access product strategy, intake journeys, and lawyer matching.

- `entities.csv`: 36 client journey records.
- `daily_metrics.csv`: 5,040 client journey-day metric rows.
- `source_events.csv`: 760 operational events, QA checks, experiments, and stakeholder asks.
- `recommended_actions.csv`: 220 candidate actions with effort and expected lift.
